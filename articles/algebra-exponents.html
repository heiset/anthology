Exponents, Exponentials, and Sequences

exponent rules, roots, exponential growth, and geometric sequences

\[ \begin{aligned} a^m \cdot a^n &= a^{m+n} &\qquad \dfrac{a^m}{a^n} &= a^{m-n} \\[0.6em] (a^m)^n &= a^{mn} &\qquad (ab)^n &= a^n b^n \\[0.6em] \left(\dfrac{a}{b}\right)^n &= \dfrac{a^n}{b^n} &\qquad a^0 &= 1 \\[0.6em] a^{-n} &= \dfrac{1}{a^n} \end{aligned} \]

Exponent Rules

Simplify \(5^{-4}\).

A negative exponent means "move to the denominator":

\[5^{-4} = \frac{1}{5^4} = \frac{1}{625}\]

Simplify \(\dfrac{8^{-1}}{(-4)^0}\).

\[\frac{8^{-1}}{(-4)^0} = \frac{8^{-1}}{1} = 8^{-1} = \frac{1}{8}\]

Simplify \(\dfrac{7^{-2}\,m^0}{n^{-4}}\).

Use \(m^0 = 1\), then move factors with negative exponents across the fraction bar:

\[\frac{7^{-2}\,m^0}{n^{-4}} = \frac{7^{-2} \cdot 1}{n^{-4}} = \frac{n^4}{7^2} = \frac{n^4}{49}\]

Simplify \(\dfrac{t^{-5}}{8^{-2}\,s^{-3}}\).

Every factor with a negative exponent moves across the bar and becomes positive:

\[\frac{t^{-5}}{8^{-2}\,s^{-3}} = \frac{8^2\,s^3}{t^5} = \frac{64s^3}{t^5}\]

Roots and Radicals

The \(n\)th root of \(a\) is the number whose \(n\)th power equals \(a\). When the index \(n\) is even and \(a\) is positive, there are two real roots (\(\pm\)). When \(n\) is odd, there is exactly one real root, and it carries the sign of \(a\).

Find the real 6th roots of 64.

\[x^6 = 64 = 2^6 \;\Rightarrow\; x = \pm 2\]

The index 6 is even and 64 is positive, so there are two real roots.

A cube has volume \(729\text{ cm}^3\). Find the side length.

\[s^3 = 729 \;\Rightarrow\; s = \sqrt[3]{729} = 9\text{ cm}\]

(Since \(9^3 = 729\).)

Evaluate \(-\sqrt[3]{-512}\).

\[\sqrt[3]{-512} = -8 \qquad \text{(since }(-8)^3 = -512\text{)}\] \[-\sqrt[3]{-512} = -(-8) = 8\]

Exponential Functions

An exponential function has the form \(y = ab^x\), where \(b > 0\) and \(b \ne 1\). The key requirement is that the variable appears in the exponent — and the base must be a genuine constant different from 1.

Is \(y = -6^x\) an exponential function?

Rewrite as \(y = -(6^x) = -1 \cdot 6^x\). This fits \(y = ab^x\) with \(a = -1\) and \(b = 6\). The variable \(x\) is in the exponent and the base 6 ≠ 1, so yes — it's exponential. The negative coefficient reflects the curve below the \(x\)-axis but doesn't change the classification.

Is \(y = 5(1)^x\) an exponential function?

\[y = 5(1)^x = 5 \cdot 1 = 5\]

Since \(1^x = 1\) for every \(x\), this is just the constant function \(y = 5\) — a horizontal line. The base must not equal 1 for the function to be truly exponential.

The green and red curves are exponential (one grows, one decays below the axis). The orange line is constant — not exponential.

Exponential Growth

An exponential growth model has the form \(f(t) = a(1+r)^t\), where \(a\) is the initial amount and \(r\) is the growth rate as a decimal.

For \(f(t) = 220(1.015)^t\), find the initial amount and growth rate. Evaluate at \(t = 5\).

\[ \begin{aligned} a &= 220 \\ 1 + r &= 1.015 \;\Rightarrow\; r = 0.015 = 1.5\% \\ f(5) &= 220(1.015)^5 \approx 237.0 \end{aligned} \]

For \(p(t) = 5.5(1.5)^t\), find the initial amount and growth rate. Evaluate at \(t = 5\).

\[ \begin{aligned} a &= 5.5 \\ 1 + r &= 1.5 \;\Rightarrow\; r = 0.5 = 50\% \\ p(5) &= 5.5(1.5)^5 \approx 41.8 \end{aligned} \]

Solving Exponential Equations

When two exponential expressions have the same base, their exponents must be equal. Rewrite both sides with the same base, then set the exponents equal and solve.

Solve \(3^{8x} = 3^{5x-6}\).

Same base on both sides, so set exponents equal:

\[ \begin{aligned} 8x &= 5x - 6 \\ 3x &= -6 \\ x &= -2 \end{aligned} \] Check: \(3^{8(-2)} = 3^{-16}\) and \(3^{5(-2)-6} = 3^{-16}\) ✓

Solve \(2^{2x} = 2^{5x+3}\).

\[ \begin{aligned} 2x &= 5x + 3 \\ -3x &= 3 \\ x &= -1 \end{aligned} \] Check: \(2^{2(-1)} = 2^{-2}\) and \(2^{5(-1)+3} = 2^{-2}\) ✓

Geometric Sequences

In a geometric sequence, each term is obtained by multiplying the previous term by a fixed number called the common ratio \(r\). To find \(r\), divide any term by the one before it.

Find the common ratio: \(5,\ 20,\ 80,\ 320,\ \ldots\)

\[r = \frac{20}{5} = 4 \qquad \text{check: } \frac{80}{20} = 4 \;\checkmark\]

Find the common ratio: \(144,\ -72,\ 36,\ -18,\ \ldots\)

\[r = \frac{-72}{144} = -\frac{1}{2} \qquad \text{check: } \frac{36}{-72} = -\frac{1}{2} \;\checkmark\]

A negative ratio causes the terms to alternate in sign.

Find the common ratio: \(24,\ 84,\ 294,\ 1029,\ \ldots\)

\[r = \frac{84}{24} = \frac{7}{2} \qquad \text{check: } \frac{294}{84} = \frac{7}{2} \;\checkmark\]

Recursive Rules and Explicit Formulas

A sequence can be described two ways. A recursive rule defines each term in terms of the one before it. An explicit formula gives the \(n\)th term directly, without computing all previous terms.

\[ \begin{array}{ll} \text{Arithmetic} & a_n = a_{n-1} + d \qquad (\text{adds the same value each step}) \\[4pt] \text{Geometric} & a_n = r \cdot a_{n-1} \qquad (\text{multiplies by the same value each step}) \end{array} \] The explicit formula for a geometric sequence is: \[a_n = a_1 \, r^{n-1}\]

Is \(a_1 = 5,\; a_n = 12a_{n-1}\) arithmetic or geometric?

The rule multiplies by 12, so it's geometric with \(r = 12\).

Is \(a_1 = 6,\; a_n = a_{n-1} - 3\) arithmetic or geometric?

The rule subtracts 3 (adds \(-3\)), so it's arithmetic with \(d = -3\).

Write the \(n\)th term for \(\dfrac{3}{125},\ \dfrac{3}{25},\ \dfrac{3}{5},\ 3,\ \ldots\) and find \(a_6\).

\[r = \frac{\;\tfrac{3}{25}\;}{\tfrac{3}{125}} = \frac{3}{25} \cdot \frac{125}{3} = 5\] \[a_n = \frac{3}{125}(5)^{n-1}\] \[a_6 = \frac{3}{125}(5^5) = \frac{3}{125}(3125) = 75\]

Write the \(n\)th term for \(0.2,\ 1.6,\ 12.8,\ 102.4,\ \ldots\) and find \(a_6\).

\[r = \frac{1.6}{0.2} = 8\] \[a_n = 0.2(8)^{n-1}\] \[a_6 = 0.2(8^5) = 0.2(32{,}768) = 6553.6\]
