The Weierstrass Function

continuous everywhere, differentiable nowhere

Hard Problems Mersenne Primes Fundamental Theorem Why 1 Is Not Prime Almost Primes Prime Number Theorem The Green-Tao Theorem Gödel's Incompleteness Furstenberg's Proof First Isomorphism Theorem The Weierstrass Function

For most of mathematical history continuity and differentiability were thought to go hand in hand. A continuous curve — one you draw without lifting your pen — was imagined as smooth enough that it had a slope almost everywhere. Then in 1872 Karl Weierstrass presented a function that demolished this picture entirely.

Continuity versus differentiability

A function \(f\) is continuous at \(x_0\) if \(\lim_{x \to x_0} f(x) = f(x_0)\). It is differentiable at \(x_0\) if the limit

\[ f'(x_0) = \lim_{h \to 0} \frac{f(x_0 + h) - f(x_0)}{h} \]

exists. Differentiability is strictly stronger: every differentiable function is continuous, but Weierstrass showed the converse is spectacularly false.

The function

\[ W(x) = \sum_{n=0}^{\infty} a^n \cos(b^n \pi x) \]

where \(0 < a < 1\), \(b\) is a positive odd integer, and \(ab > 1 + \tfrac{3\pi}{2}\). The parameter \(a\) controls how fast amplitudes shrink (\(a^n \to 0\)); the parameter \(b\) controls how rapidly oscillations speed up (each term oscillates \(b\) times faster than the last). The tension between these forces creates a function that is continuous (the shrinking amplitudes prevent gaps) but has no slope anywhere (the accelerating oscillations prevent any straight-line approximation).

The function is a fractal: zooming in at any point reveals the same jagged complexity, never resolving into a line. Hermite called such functions "a lamentable scourge"; Poincaré called them "an affront to good sense." Today we recognize them as fundamental — Brownian motion, the random wandering of pollen grains in water, is modelled by functions with exactly this character.

Challenge problem

(a) Prove from the definition of the derivative that \(f(x) = |x|\) is not differentiable at \(x = 0\). Compute the left-hand limit \(\lim_{h \to 0^-} |h|/h\) and right-hand limit \(\lim_{h \to 0^+} |h|/h\) separately and show they disagree.

(b) Let \(g(x) = x\sin(1/x)\) for \(x \neq 0\) and \(g(0) = 0\). Prove \(g\) is continuous at \(x = 0\) using the Squeeze Theorem (\(-|x| \leq x\sin(1/x) \leq |x|\)). Then prove \(g\) is not differentiable at 0 by showing the difference quotient \(g(x)/x = \sin(1/x)\) has no limit as \(x \to 0\).

(c) In Weierstrass's original construction, verify that \(a = 0.9\) and \(b = 7\) satisfy the condition \(ab > 1 + 3\pi/2 \approx 5.712\). Explain informally why, if \(a\) were very small (fast amplitude decay) and \(b\) very large (fast oscillation), you might actually get a differentiable function — so the condition \(ab > 1 + 3\pi/2\) is essential.
