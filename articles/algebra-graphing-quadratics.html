Graphing Quadratics

transformations, vertex form, completing the square, and factored form

\[y = a(x-h)^2 + k\]

Vertex \((h,k)\) · axis \(x = h\) · opens up if \(a > 0\), down if \(a < 0\) · narrower than \(y=x^2\) if \(|a|>1\), wider if \(|a|<1\)

Transformations of \(y = x^2\)

Every parabola is a transformed version of the parent graph \(y = x^2\). Adding or subtracting a constant shifts it; changing the coefficient stretches or compresses it; negating it flips it upside down. You can read all of this directly from vertex form.

Change to \(y=x^2\) Effect on graph Example
\(+k\)shift up \(k\)\(x^2+3\)
\(-k\)shift down \(k\)\(x^2-5\)
\((x-h)^2\)shift right \(h\)\((x-2)^2\)
\((x+h)^2\)shift left \(h\)\((x+4)^2\)
\(a > 1\)narrower\(3x^2\)
\(0 < a < 1\)wider\(\tfrac14 x^2\)
\(a < 0\)reflect over \(x\)-axis\(-x^2\)

Graph \(h(x) = x^2 + 10\).

Start with \(y = x^2\), then shift every point up 10 units. The vertex moves from \((0,0)\) to \((0,10)\); the shape and width are unchanged.

Graph \(h(x) = -\tfrac{1}{4}x^2 - 1\).

Apply transformations step by step:

\[ \begin{aligned} y &= x^2 && \text{parent} \\ y &= \tfrac{1}{4}x^2 && \text{vertically compress (wider)} \\ y &= -\tfrac{1}{4}x^2 && \text{reflect over } x\text{-axis} \\ y &= -\tfrac{1}{4}x^2 - 1 && \text{shift down 1} \end{aligned} \] Vertex: \((0,-1)\). Opens downward. Wider than \(y = x^2\).

Completing the Square

Any quadratic can be rewritten in vertex form \(y = a(x-h)^2+k\) by completing the square. Once there, the vertex, axis, direction, and width all read off immediately — and graphing by hand becomes straightforward.

Rewrite \(y = -5x^2 - 20x + 4\) in vertex form and graph.

Factor \(-5\) out of the quadratic and linear terms only:

\[y = -5(x^2 + 4x) + 4\]

Complete the square inside the parentheses. Half of 4 is 2, and \(2^2 = 4\):

\[x^2 + 4x = (x+2)^2 - 4\]

Substitute back:

\[ \begin{aligned} y &= -5\bigl((x+2)^2 - 4\bigr) + 4 \\ &= -5(x+2)^2 + 20 + 4 \\ &= -5(x+2)^2 + 24 \end{aligned} \]

Now read off everything:

  • Vertex: \((-2, 24)\)
  • Axis of symmetry: \(x = -2\)
  • Since \(a = -5 < 0\), opens downward
  • Since \(|a| = 5 > 1\), narrower than \(y = x^2\)

To get easy plotting points, move 1 unit left and right from the vertex. Since \(a = -5\), the graph drops by 5:

\[(-1,\,19),\quad (-3,\,19)\]

The \(y\)-intercept at \(x = 0\): \(y = -5(0)^2 - 20(0) + 4 = 4\). By symmetry, \((-4, 4)\) is also on the graph.

Graphing from Factored Form

When a quadratic is already in factored form, the zeros come directly from the zero product property, the axis of symmetry is the average of the zeros, and substituting the axis value gives the vertex.

Find the vertex of \(f(x) = -\tfrac{1}{3}x(x+5)\).

Zeros: set each factor to zero — \(x = 0\) or \(x = -5\).

\[\text{Axis of symmetry: } x = \frac{0 + (-5)}{2} = -\frac{5}{2}\] \[f\!\left(-\tfrac{5}{2}\right) = -\tfrac{1}{3}\left(-\tfrac{5}{2}\right)\!\left(-\tfrac{5}{2}+5\right) = -\tfrac{1}{3}\cdot\left(-\tfrac{5}{2}\right)\cdot\tfrac{5}{2} = -\tfrac{1}{3}\cdot\left(-\tfrac{25}{4}\right) = \frac{25}{12}\] \[\text{Vertex: }\left(-\tfrac{5}{2},\;\frac{25}{12}\right)\]

Note the sign: \(-\tfrac{1}{3} \cdot (-\tfrac{25}{4})\) is positive, so the vertex is a maximum — which makes sense since \(a = -\tfrac{1}{3} < 0\) means the parabola opens downward.

Graph \(f(x) = 4(x+3)(x+2)\).

Zeros: \(x = -3\) and \(x = -2\). Axis: \(x = \tfrac{-3+(-2)}{2} = -\tfrac{5}{2}\).

\[f\!\left(-\tfrac{5}{2}\right) = 4\left(-\tfrac{5}{2}+3\right)\!\left(-\tfrac{5}{2}+2\right) = 4\cdot\tfrac{1}{2}\cdot\left(-\tfrac{1}{2}\right) = -1\] Key points: \((-3,0),\;(-2,0),\;\bigl(-\tfrac{5}{2},-1\bigr)\).

Find the zeros of \(f(x) = \tfrac{2}{3}(x+8)(x-5)\).

\[\tfrac{2}{3}(x+8)(x-5) = 0 \;\Rightarrow\; x = -8 \text{ or } x = 5\]

Graph \(f(x) = -2(x-5)(x-3)\).

Zeros: \(x = 3\) and \(x = 5\). Axis: \(x = \tfrac{3+5}{2} = 4\).

\[f(4) = -2(4-5)(4-3) = -2(-1)(1) = 2\] Key points: \((3,0),\;(5,0),\;(4,2)\). The parabola opens downward since \(a = -2 < 0\).

Even and Odd Functions

Functions can have algebraic symmetry that matches visual symmetry on the graph.

A function is even if \(f(-x) = f(x)\) for all \(x\) — substituting \(-x\) gives the same value as \(x\). Graphically, even functions are symmetric about the \(y\)-axis. Example: \(f(x) = x^2\), since \((-x)^2 = x^2\).

A function is odd if \(f(-x) = -f(x)\) — substituting \(-x\) negates the output. Graphically, odd functions have 180° rotational symmetry about the origin. Example: \(f(x) = x^3\), since \((-x)^3 = -x^3\).

A quadratic \(f(x) = ax^2 + bx + c\) is even only when \(b = 0\) — that is, when there's no horizontal shift and the axis of symmetry is the \(y\)-axis. The moment you shift the parabola left or right, the \(y\)-axis symmetry is lost and the function is neither even nor odd.
