The First Isomorphism Theorem

every homomorphism hides an isomorphism inside it

Hard Problems Mersenne Primes Fundamental Theorem Why 1 Is Not Prime Almost Primes Prime Number Theorem The Green-Tao Theorem Gödel's Incompleteness Furstenberg's Proof First Isomorphism Theorem The Weierstrass Function

In mathematics two structures are "the same" if there is a perfect structure-preserving bijection between them — an isomorphism. The First Isomorphism Theorem says something deeper: whenever you have any structure-preserving map (even a non-bijective one), an isomorphism is hiding inside it.

Setup

Let \(\phi: G \to H\) be a group homomorphism — a function between groups satisfying \(\phi(ab) = \phi(a)\phi(b)\) for all \(a, b \in G\). Two objects arise naturally:

\[ \ker \phi = \{g \in G : \phi(g) = e_H\}, \qquad \mathrm{im}\,\phi = \{\phi(g) : g \in G\}. \]

The kernel is a normal subgroup of \(G\), so we can form the quotient group \(G/\ker\phi\) whose elements are cosets \(g\ker\phi\).

The theorem

First Isomorphism Theorem. For any group homomorphism \(\phi: G \to H\):

\[ G\,/\ker\phi \;\cong\; \mathrm{im}\,\phi. \]

Intuition. Think of \(\phi\) as sorting elements of \(G\) into piles according to where they land in \(H\). Two elements share a pile if and only if they map to the same place. The kernel is the pile that lands on \(e_H\). Every other pile is a coset of \(\ker\phi\). The quotient \(G/\ker\phi\) is the collection of all these piles — and the theorem says the piles correspond exactly and perfectly to the elements of \(\mathrm{im}\,\phi\).

Example. Let \(\phi: \mathbb{Z} \to \mathbb{Z}/6\mathbb{Z}\) be the map \(\phi(n) = n \bmod 6\). Then \(\ker\phi = 6\mathbb{Z}\) and \(\mathrm{im}\,\phi = \mathbb{Z}/6\mathbb{Z}\). The theorem gives \(\mathbb{Z}/6\mathbb{Z} \cong \mathbb{Z}/6\mathbb{Z}\) — true but trivial here. The power is that it identifies the quotient structure automatically.

Challenge problem

(a) Define \(\psi: \mathbb{Z}/12\mathbb{Z} \to \mathbb{Z}/4\mathbb{Z}\) by \(\psi([n]_{12}) = [n]_4\). Verify \(\psi\) is well-defined and is a group homomorphism.

(b) Find \(\ker\psi\) and \(\mathrm{im}\,\psi\). List the elements of the kernel explicitly.

(c) Apply the First Isomorphism Theorem to identify \((\mathbb{Z}/12\mathbb{Z})/\ker\psi\) as a familiar group. Write out all cosets of \(\ker\psi\) in \(\mathbb{Z}/12\mathbb{Z}\).

(d) How many distinct group homomorphisms are there from \(\mathbb{Z}/6\mathbb{Z}\) to \(\mathbb{Z}/4\mathbb{Z}\)? For each, identify the kernel and image.
