Gödel's Incompleteness Theorems

mathematics cannot prove everything that is true

In 1900 David Hilbert proposed a grand program: find a complete, consistent set of axioms from which all of mathematical truth could be derived. Thirty years later Kurt Gödel proved this was impossible — not just practically, but in principle.

The theorems

First Incompleteness Theorem. Any consistent formal system powerful enough to express basic arithmetic contains true statements that cannot be proved within that system.

Second Incompleteness Theorem. No such system can prove its own consistency.

The proof idea

Gödel made arithmetic talk about itself. The key idea is Gödel numbering: assign each symbol in the formal language a natural number, then encode entire formulas as numbers via prime factorization. For example, if symbols are coded 1–12, the statement \(0 = 0\) might be encoded as

\[ 2^6 \cdot 3^5 \cdot 5^6. \]

Statements about formulas (metamathematics) now become statements about numbers (arithmetic), expressible inside the formal system itself.

With this encoding Gödel constructed a sentence \(G\) that, decoded, says: "This sentence has no proof." Then:

  • If \(G\) is provable, the system just proved a false statement — it is inconsistent.
  • If \(G\) is not provable, then \(G\) is true — and the system is incomplete.

Either way, a consistent system cannot be complete. The critical move — replacing "false" with "unprovable" — is what separates this from the Liar's Paradox ("This statement is false"). The Liar has no truth value; Gödel's sentence has a definite truth value, it simply has no proof.

Challenge problem

(a) The Liar's Paradox says "This statement is false." Suppose it is true — derive a contradiction. Suppose it is false — derive another. Show it is genuinely neither.

(b) Gödel replaced "false" with "unprovable." Show that if Gödel's sentence \(G\) is unprovable, no contradiction arises — \(G\) can simply be true. This is what separates Gödel from the Liar.

(c) The Second Theorem says a consistent system \(S\) cannot prove its own consistency. Here is a sketch of why: if \(S\) proved its own consistency, it would know any provable statement is true, and in particular it would be able to prove that \(G\) is unprovable — which is exactly what \(G\) says, so \(S\) would have proved \(G\). Why is this a contradiction? Write up the full argument in your own words.
