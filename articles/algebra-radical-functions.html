Radical Functions and Inverses

square roots, cube roots, solving radical equations, and inverse functions

Square root: radicand \(\ge 0\) · range \([0,\infty)\). Cube root: all reals · range \((-\infty,\infty)\).

To solve a radical equation: isolate the radical, raise both sides to the appropriate power, then check for extraneous solutions.

Square Root Functions

For \(y = \sqrt{\text{radicand}}\), the radicand must be nonnegative. Set it \(\ge 0\) and solve to find the domain. The output of a square root is always nonnegative, so the range starts at 0.

Domain of \(y = \sqrt{7x}\).

\[7x \ge 0 \;\Rightarrow\; x \ge 0 \qquad \text{Domain: } [0,\infty)\]

Domain of \(y = \sqrt{-x+5}\).

\[-x + 5 \ge 0 \;\Rightarrow\; -x \ge -5\]

Dividing (or multiplying) by a negative flips the inequality sign:

\[x \le 5 \qquad \text{Domain: } (-\infty, 5]\]

Domain and range of \(y = 3\sqrt{-x}\).

\[ -x \ge 0 \;\Rightarrow\; x \le 0 \qquad \text{Domain: } (-\infty, 0] \] \[ \sqrt{-x} \ge 0 \;\Rightarrow\; 3\sqrt{-x} \ge 0 \qquad \text{Range: } [0,\infty) \]

Graph \(g(x) = \sqrt{\tfrac{x}{3}}\) alongside \(f(x) = \sqrt{x}\).

Replacing \(x\) with \(\tfrac{x}{3}\) stretches the graph horizontally by a factor of 3. You need to go three times as far along the \(x\)-axis to get the same output.

\[\text{Domain: } [0,\infty) \qquad \text{Range: } [0,\infty)\]

Graph \(g(x) = -\sqrt{x} + 2\) alongside \(f(x) = \sqrt{x}\).

The negative reflects the graph across the \(x\)-axis; the \(+2\) shifts it up. The function decreases from \((0,2)\) downward.

\[\text{Domain: } [0,\infty) \qquad \text{Range: } (-\infty, 2]\]

Cube Root Functions

Cube roots don't require a nonnegative radicand — you can cube-root any real number. So cube root functions have domain and range \((-\infty,\infty)\), and their graphs pass through the origin and extend in both directions.

Graph \(g(x) = \sqrt[3]{x+4}\) alongside \(f(x) = \sqrt[3]{x}\).

Replacing \(x\) with \(x+4\) shifts the graph left 4. The center of the S-curve moves from \((0,0)\) to \((-4,0)\).

\[\text{Domain: } (-\infty,\infty) \qquad \text{Range: } (-\infty,\infty)\]

Graph \(g(x) = -\sqrt[3]{x-3}\) alongside \(f(x) = \sqrt[3]{x}\).

The \(x-3\) inside shifts the graph right 3. The leading negative reflects it across the \(x\)-axis — the curve that was rising from left to right now falls.

\[\text{Domain: } (-\infty,\infty) \qquad \text{Range: } (-\infty,\infty)\]

Solving Radical Equations

Isolate the radical, then undo it by squaring (for square roots) or cubing (for cube roots). Squaring can create fake solutions — always check your answers in the original equation.

Solve \(\sqrt{x} = 13\).

\[(\sqrt{x})^2 = 13^2 \;\Rightarrow\; x = 169\]

Solve \(-16 = 4 - \sqrt{y}\).

\[ \begin{aligned} -20 &= -\sqrt{y} \\ 20 &= \sqrt{y} \\ 400 &= y \end{aligned} \]

Solve \(\sqrt{q-4} - 10 = -7\).

\[ \begin{aligned} \sqrt{q-4} &= 3 \\ q-4 &= 9 \\ q &= 13 \end{aligned} \]

Solve \(3 = \sqrt{2x+2} - 9\).

\[ \begin{aligned} 12 &= \sqrt{2x+2} \\ 144 &= 2x+2 \\ x &= 71 \end{aligned} \]

Solve \(\sqrt{3x} = \sqrt{4x-6}\).

Square both sides directly:

\[3x = 4x - 6 \;\Rightarrow\; x = 6\]

Solve \(\sqrt[3]{x-9} = 2\).

Cube both sides (no extraneous solutions with cube roots):

\[x - 9 = 8 \;\Rightarrow\; x = 17\]

Solve \(\sqrt{4t+24} = -2t\).

\[ \begin{aligned} 4t + 24 &= 4t^2 \\ 0 &= 4t^2 - 4t - 24 \\ 0 &= t^2 - t - 6 \\ 0 &= (t-3)(t+2) \\ t &= 3 \text{ or } t = -2 \end{aligned} \]

Check both — this is where squaring can create trouble:

\[t = 3: \quad \sqrt{36} = 6 \;\ne\; -6 \qquad \text{extraneous}\] \[t = -2: \quad \sqrt{16} = 4 = -2(-2) = 4 \qquad \checkmark\] \[\boxed{t = -2}\]

Inverse Functions

An inverse relation swaps inputs and outputs. To find an inverse function, write \(y = f(x)\), swap \(x\) and \(y\), then solve for \(y\). The graphs of \(f\) and \(f^{-1}\) are reflections of each other across the line \(y = x\).

Find the inverse of the relation.

Input −10 −5 0 5 10 15
Output 1 2 1 2 1 2

Switch every input and output:

Input 1 2 1 2 1 2
Output −10 −5 0 5 10 15

Notice the inverse is not a function — input 1 maps to three different outputs. The original relation failed the horizontal line test.

Find the inverse of \(f(x) = 4x + 5\). What input gives output 2?

To find the inverse relation, start from \(y = 4x+5\) and solve for \(x\) in terms of \(y\):

\[y - 5 = 4x \;\Rightarrow\; x = \frac{y-5}{4}\]

So if the output is 2:

\[x = \frac{2-5}{4} = -\frac{3}{4}\]

Find \(f^{-1}(x)\) for \(f(x) = 3x - 4\).

\[ \begin{aligned} y &= 3x - 4 \\ x &= 3y - 4 && \text{swap } x \text{ and } y \\ x + 4 &= 3y \\ \frac{x+4}{3} &= y \end{aligned} \] \[f^{-1}(x) = \frac{x+4}{3}\]

The graph shows \(f\) and \(f^{-1}\) as mirror images across the line \(y = x\).

The Horizontal Line Test

A function has an inverse that is also a function if and only if every horizontal line crosses the original graph at most once.

Does \(f(x) = \sqrt{x+2} + 3\) pass the Horizontal Line Test?

This is a square root function shifted left 2 and up 3. It always increases (moving right, the output grows), so each horizontal level is reached at most once.

\[\text{Passes the Horizontal Line Test} \;\Rightarrow\; f^{-1} \text{ is a function.}\]
