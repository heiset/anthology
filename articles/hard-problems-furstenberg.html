Furstenberg's Topological Proof

infinitely many primes, proved with open sets

Hard Problems Mersenne Primes Fundamental Theorem Why 1 Is Not Prime Almost Primes Prime Number Theorem The Green-Tao Theorem Gödel's Incompleteness Furstenberg's Proof First Isomorphism Theorem The Weierstrass Function

That there are infinitely many primes has been known since Euclid (~300 BCE). His proof is a one-paragraph gem. But in 1955 a 20-year-old Hillel Furstenberg published a proof using topology — the branch of mathematics dealing with abstract notions of openness and continuity, typically far removed from elementary number theory.

The topology

Define a topology on \(\mathbb{Z}\) by declaring that a set \(U \subseteq \mathbb{Z}\) is open if it is either empty or a union of two-way arithmetic progressions

\[ S(a, d) = \{a + nd : n \in \mathbb{Z}\} = \{\ldots,\; a - 2d,\; a - d,\; a,\; a + d,\; a + 2d,\; \ldots\} \]

where \(a \in \mathbb{Z}\) and \(d \in \mathbb{Z}^+\). One can verify this defines a valid topology on \(\mathbb{Z}\).

Key observation: each \(S(a, d)\) is both open and closed (clopen). It is open by definition. Its complement is

\[ S(a,d)^c = \bigcup_{r=1}^{d-1} S(a+r,\, d), \]

a finite union of open sets, hence open. So \(S(a, d)\) is closed.

The proof

Every integer except \(\pm 1\) has at least one prime divisor, so

\[ \mathbb{Z} \setminus \{-1, 1\} = \bigcup_{p \text{ prime}} S(0, p). \]

Each \(S(0, p)\) is closed. If there were finitely many primes, this union would be finite — hence closed. Then \(\{-1, 1\}\) would be open.

But in this topology every non-empty open set is infinite: it must contain an arithmetic progression. The set \(\{-1, 1\}\) is finite and non-empty, so it cannot be open. Contradiction. \(\square\)

Challenge problem

(a) Prove that in Furstenberg's topology every non-empty open set is infinite.

(b) Prove that the complement of \(S(a, d)\) is a finite union of arithmetic progressions, and therefore open. Conclude \(S(a, d)\) is closed.

(c) Write out Euclid's proof in full. Then identify the structural parallel: in both Euclid's and Furstenberg's proofs, what is the key object that leads to the contradiction, and how do the two proofs differ in constructing it?
