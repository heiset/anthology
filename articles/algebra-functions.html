Functions and Lines

the building blocks of algebra

Functions, Domain, and Range

A function is a rule that takes an input and produces exactly one output — each input has only one result. The easiest way to check: if the same input ever gives two different outputs, it's not a function.

Given the ordered pairs \((-4,2),\ (2,-4),\ (6,6),\ (12,18),\ (18,12)\), each input (\(x\)-value) appears exactly once, so this is a function.

For a graph, use the vertical line test: if any vertical line crosses the graph more than once, the relation is not a function — one \(x\)-value would produce two different \(y\)-values.

The domain is all the \(x\)-values the function accepts. The range is all the \(y\)-values it can produce. Read domain by scanning left to right, range bottom to top.

\[\text{Domain: } [-1,\,3] \qquad \text{Range: } [-8,\,8]\]

When a graph has gaps — pieces that skip over certain values — domain and range must be listed piece by piece:

\[ \begin{aligned} \text{Domain:} &\quad 1 \le x < 2,\quad 2 < x < 4,\quad 4 < x \le 6 \\[4pt] \text{Range:} &\quad 1 \le y < 3,\quad 3 < y < 5,\quad 5 < y \le 6 \end{aligned} \]

Zeros and Intercepts

The zeros (or intercepts) of a function are the \(x\)-values where it crosses the \(x\)-axis — the places where \(y = 0\). They're called zeros because the output is literally zero there.

\[\text{Example intercepts: }\bigl(-\tfrac{5}{2},0\bigr),\;(0,0),\;(1,0)\]

Two more useful ideas when reading a graph:

  • A function is positive where it lives above the \(x\)-axis, and negative where it dips below.
  • A function is increasing (\(\nearrow\)) when larger \(x\) produces larger \(y\); decreasing (\(\searrow\)) when larger \(x\) produces smaller \(y\).
\[ \begin{aligned} \textbf{Positive:} &\quad 1 < x < 3 \\[2pt] \textbf{Negative:} &\quad x < 1 \text{ and } x > 3 \\[2pt] \textbf{Increasing:} &\quad -\infty < x < 2 \\[2pt] \textbf{Decreasing:} &\quad 2 < x < \infty \end{aligned} \]

Linear Functions

A linear function has only plain \(x\) and \(y\) — no exponents, no square roots, no \(y^2\). Multiply by fractions or negatives all you want; just keep the variables simple.

Consider the table:

\(x\) 0 2 4 6
\(y\) 3 9 27 81

The \(x\)-values increase by 2 each step. But the \(y\)-values go \(3 \to 9 \to 27 \to 81\) — each multiplied by 3, not increased by a fixed amount. That's exponential growth, not linear.

Compare \(y - \frac{1}{3}x = 4x - 7\): rearranging gives \(y = \frac{13}{3}x - 7\) — plain \(x\) and \(y\), so this is linear. But \((y+2)(y-4) = 3x\) expands to \(y^2 - 2y - 8 = 3x\), and that \(y^2\) makes it non-linear.

Evaluating and Solving

To evaluate a function, substitute a specific number in for \(x\). For \(f(x) = 1.5x + 1\):

\[ \begin{aligned} f(-2) &= 1.5(-2)+1 = -3+1 = -2 \\[2pt] f(0) &= 1.5(0)+1 = 1 \\[2pt] f(5) &= 1.5(5)+1 = 8.5 \end{aligned} \]

To solve, you're given the output and asked to find the input. For \(f(x) = 8x - 7\), find \(x\) when \(f(x) = 17\):

\[ \begin{aligned} 17 &= 8x - 7 \\[2pt] 24 &= 8x \\[2pt] x &= 3 \end{aligned} \]

Graphing with Intercepts

The fastest way to graph a line is to find its two intercepts and connect them.

\(y = 1\) is a horizontal line — every point has \(y = 1\) regardless of \(x\). If it were \(x = 1\), the line would be vertical.

For \(-5x + 7y = -35\), find both intercepts by setting the other variable to zero:

\(x\)-intercept (set \(y = 0\)):

\[-5x = -35 \implies x = 7 \quad\Longrightarrow\quad (7,\,0)\]

\(y\)-intercept (set \(x = 0\)):

\[7y = -35 \implies y = -5 \quad\Longrightarrow\quad (0,\,-5)\]

Slope

Slope measures how steeply a line rises or falls: \(m = \dfrac{\Delta y}{\Delta x}\), rise over run. A vertical line has undefined slope (zero run); a horizontal line has slope zero.

When a table shows the same \(x\)-value for every row, the points form a vertical line — no slope, and the equation is just \(x = c\). For example:

\(x\) 3 1 −1 −3
\(y\) −4 1 6 11

Pick any two points — say \((3,-4)\) and \((1,1)\):

\[m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{1-(-4)}{1-3} = \frac{5}{-2} = -\frac{5}{2}\]

To find slope from a standard-form equation, rewrite in slope-intercept form \(y = mx + b\). For \(-4x = 9 - 2y\):

\[2y = 4x + 9 \implies y = 2x + \frac{9}{2}\]

slope \(m = 2\), \(y\)-intercept \(b = \frac{9}{2}\).

Full example — \(-3x + y = 7\). Find the intercepts, slope, and graph.

\[ \begin{aligned} x\text{-intercept:} &\quad y=0 \implies -3x = 7 \implies x = -\tfrac{7}{3} \\[4pt] y\text{-intercept:} &\quad x=0 \implies y = 7 \end{aligned} \]

Rewriting as \(y = 3x + 7\) gives slope \(m = 3\): every step right moves the line up 3.
